# Gerenciador de Tarefas

Sistema fullstack de gerenciamento de tarefas com autenticação JWT.

## Tecnologias
- **Backend:** Node.js + TypeScript + Express + PostgreSQL
- **Frontend:** React + TypeScript + Vite

## Como rodar

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
